# PowerFactory Automation Scripts

Repositorio para organizar scripts que automaticen procesos en **DIgSILENT PowerFactory**.

## Objetivo
- Centralizar scripts de automatización.
- Documentar flujos de trabajo repetitivos.
- Compartir ejemplos reutilizables.

## Estructura
- `scripts/`: scripts de automatización (Python/DPL u otros).
- `docs/`: documentación de flujos, requisitos y guías.

## Primeros pasos
1. Agrega tus scripts en `scripts/`.
2. Documenta el propósito y cómo ejecutarlos en `docs/`.

## Convenciones sugeridas
- Nombres descriptivos: `exportar_resultados.py`, `actualizar_modelo.dpl`.
- Incluir encabezado con descripción, versión y responsable.

## Requisitos
- PowerFactory instalado.
- Entorno de scripting habilitado (Python API o DPL).
